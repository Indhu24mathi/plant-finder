# 🌱 Soil Pollution Detection & Phytoremediation Plant Selector

An AI-powered web application that predicts soil contamination 
levels and recommends suitable plants for soil restoration.

## 🚀 Features
- Predicts soil pollution level using a trained ANN model
- Recommends phytoremediation plants based on soil data
- Real-time dashboard with soil health insights
- Built with Django + TensorFlow

## 🛠️ Tech Stack
Python | Django | TensorFlow | SQLite | HTML | CSS

## 📸 How It Works
1. Enter soil parameters (pH, moisture, temperature)
2. AI model analyzes contamination level
3. System recommends the best plants for soil restoration

## ▶️ Run Locally
```bash
pip install django tensorflow numpy scikit-learn
python manage.py runserver
```
Then open: http://127.0.0.1:8000
